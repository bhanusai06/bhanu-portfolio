# Bhanu Sai Veera Ashok Babu Sonti — Portfolio

## Files
- `index.html` — The complete portfolio website (single file)
- `Bhanu_Resume.pdf` — Resume (linked to the Download Resume button)
- `vercel.json` — Vercel deployment config

## Deploy on Vercel (60 seconds)
1. Go to https://vercel.com/new
2. Click "Deploy without Git" → drag this entire folder
3. Click Deploy — done ✅

## Deploy on Netlify (60 seconds)
1. Go to https://app.netlify.com/drop
2. Drag this entire folder onto the page — done ✅

## Deploy on GitHub Pages
1. Create a new repo on GitHub
2. Upload all files
3. Settings → Pages → Source: main branch → Save
4. Your site: https://bhanusai06.github.io/portfolio

## Update Resume
Replace `Bhanu_Resume.pdf` with your new PDF (keep the same filename).
